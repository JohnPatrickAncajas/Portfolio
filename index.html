Experience!

  • Portfolio (2023 - 2024)

    A project made using HTML, CSS, and JavaScript with the help of Bootstrap library and Vue framework.


    HTML forms the structural foundation, providing the layout and content structure. CSS adds styling and aesthetics, ensuring a visually pleasing presentation. JavaScript brings interactivity to the portfolio, making it engaging and dynamic.


    The Bootstrap library plays a crucial role by offering responsive design components and pre-built styles. This ensures that the portfolio looks and functions seamlessly across different devices and screen sizes.

    Vue.js takes interactivity to the next level. It enables real-time updates and dynamic features, enhancing the user experience. With Vue.js, I can create interactive elements like modals, accordions, and dynamic content loading, making the portfolio both intuitive and efficient.

    In essence, my portfolio is a harmonious fusion of these technologies. It's not just a showcase of my work; it's a testament to the power of HTML, CSS, JavaScript, Bootstrap, and Vue.js in crafting engaging and user-friendly web experiences.
