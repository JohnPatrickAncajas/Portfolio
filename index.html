My Experience!

  • Portfolio (2023 - 2024)

    A project made using the front-end development stack with Bootstrap and Vue.


    HTML forms the structural foundation, providing the layout and content structure. CSS adds styling and aesthetics, ensuring a visually pleasing presentation. JavaScript brings interactivity to the portfolio, making it engaging and dynamic.


    The Bootstrap library plays a crucial role by offering responsive design components and pre-built styles. This ensures that the portfolio looks and functions seamlessly across different devices and screen sizes.

    Vue.js takes interactivity to the next level. It enables real-time updates and dynamic features, enhancing the user experience. With Vue.js, I can create interactive elements like modals, accordions, and dynamic content loading, making the portfolio both intuitive and efficient.

  • Addventure (2023 - 2024)

    A capstone project designed to be an educational game made using HTML, CSS and JS.


    I created a capstone project which is an educational game called Addventure that teaches grade 1 math skills. The game is made using HTML, CSS, and JavaScript, and it covers a wide range of grade 1 math topics but mostly focuses in addition. The game is designed to be challenging but achievable, and players are rewarded for their correct answers with positive feedback and fun animations.

  • FARO (2023 - 2024)

    A fantasy adventure game created using HTML, CSS and JavaScript.


    In this simple yet engaging fantasy adventure game created with HTML, CSS, and JavaScript, players find themselves in a world teeming with fictional monsters. Assuming the role of a valiant knight, they engage in turn-based battles. As players progress through the story, they must strategically manage their limited resources, including healing potions and escape ropes to overcome challenges and add depth to this simple yet immersive adventure.
